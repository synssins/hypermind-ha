# Hypermind Integration for Home Assistant

Monitor your [Hypermind](https://github.com/lklynet/hypermind) P2P deployment counter in Home Assistant.

## Sensors

- **Active Nodes**: Total Hypermind instances running worldwide
- **Direct Connections**: Your direct peer connections

## Example Use Case

Create an automation to change RGBW light colors based on node count - green when low, red when high!

## Requirements

- A running Hypermind instance
- Network access from Home Assistant to Hypermind
